# Процесс создания личного сайта
***Цель***: создать персональный сайт айтишника с библиотекой, 
менеджером задач, календарем событий, вики (базой знаний) 
и проектами. 

Сайт выступает портфолио и демонстрирует степень владения 
фреймворком **Django** и REST API со сторонними сервисами, 
такими как **Yandex 360**, **Telegram API**. 

Разработка ведется от простого к сложному.

Используется **bootstrap 5**.

Интерфейс генерируется **deepseek**.

## Задачи:
* Безопасность и лучшие практики
* Шаблоны
* Отправка почты с сайта или формы на Yandex360
* SEO
* Разные приложения и связь между ними
* Толстые и тонкие модели, декомпозиция, когда это уместно.
* CI/CD


## Этапы разработки
Редактировать по мере разработки. Оставлять комменты 
согласно этапа. 
1. <u>Создать простой одностраничник с минимальным функционалом.</u>
    1.1 [x] Создать первоначальный шаблон проекта Django.</u>

        * Шаблон проекта позволит автоматизировать создание новых проектов в VSC
        * сверстанные страницы сгенерированные DeepSeek
        * проект и его запуск
        * защитить критичную информацию с помощью django-environ
        * инициализировать git и положить в репозитарий

    **Вывод** создали проект, все проекты лучше называть core,
    для хранения критичной информации лучше использовать отдельный 
    .env файл и библиотеку django-environ. Инициализировали наш шаблон в git как первый коммент и публикация в репозитарий без критичной информации.

    1.2 [x] Отображение HTML страницы и шаблоны в проекте
        * Простое отображение страницы в проекте, папка templates
        * декомпозиция HTML, на css и js, статичные файлы проекта.
        * декомпозиция HTML на отдельные секции, мощь шаблонов

    **Вывод** на уровне проекта показали HTML файл, выделили статичные файлы стилей и js. Декомпозировали саму страницу 
    на отдельные смысловые блоки. Изучили понятие шаблона, шаблонный тэг static. Что такое статичные файлы и как их эмулирует среда разработки. Понятие декомпозиция.

    1.3 [ ] <u>Приложение и проект. Первое приложение.</u>
        * создание приложения home для главной страницы или дашбоарда с шаблонами хедера и футера.
        * структура 
    1.4 [ ] Отправка почты с сайта или заполнение форм на Yandex360
    1.5. [ ] Полнофункциональный лендинг c (версия 1.0)
2. Добавить приложение менеджер задач
